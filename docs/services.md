# Dictionary Service For FastWQ

## Local

  - **MdxService**, .mdx dict file, it's only has default field.
  - **StardictService**, .ifo dict file, it's only has default field.
  - **MDX-LDOCE6**, local LDOCE6 mdx file, it's has some fileds.
